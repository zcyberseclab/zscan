# Ollama 漏洞

## 产品简介
Ollama 是一个流行的开源 AI 模型运行框架，用于本地部署和运行大语言模型（LLM）。

## 漏洞列表

### 1. ollama-unauthaccess - 未授权访问
- **危害等级**: 高危
- **漏洞描述**: Ollama 默认不支持身份验证，暴露在互联网上的实例可被任意访问，攻击者可以查看、下载、修改 AI 模型。
- **影响版本**: 所有版本
- **修复建议**: 使用反向代理添加认证，不要将 Ollama 直接暴露到公网。

### 2. CVE-2024-37032 - 远程代码执行 (Probllama)
- **危害等级**: 严重
- **漏洞描述**: 由于 /api/pull 端点对 digest 字段验证不足，存在路径遍历漏洞，可导致任意文件写入，进而实现远程代码执行。
- **影响版本**: < 0.1.34
- **修复建议**: 升级到 0.1.34 或更高版本。

### 3. CVE-2024-39721 - 拒绝服务 (DoS)
- **危害等级**: 高危
- **漏洞描述**: CreateModelHandler 函数使用 os.Open 读取文件直到完成，攻击者可通过指向 /dev/random 等无限数据源造成资源耗尽。
- **影响版本**: < 0.1.34
- **修复建议**: 升级到 0.1.34 或更高版本。

### 4. CVE-2024-39722 - 文件存在性泄露
- **危害等级**: 中危
- **漏洞描述**: 攻击者可探测服务器上文件是否存在。
- **影响版本**: < 0.1.34
- **修复建议**: 升级到 0.1.34 或更高版本。
